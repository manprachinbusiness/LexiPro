idempotent
/ˌaɪdəmˈpoʊtənt/
adjective
Tap to reveal definition →
Denoting an operation that produces the same result regardless of how many times it is executed.
Example "Making the same API request multiple times should be idempotent — the second call produces the same state as the first."
💡 Context Critical in distributed systems and RESTful API design, especially for PUT and DELETE operations.
